## 🚀 स्थापना (Installation)

### 1. Repository Clone करें
```bash
git clone https://github.com/YOUR_USERNAME/ayurveda-pregnancy-app.git
cd ayurveda-pregnancy-app
```

### 2. Dependencies Install करें
```bash
pip install -r requirements.txt
```

### 3. App चलाएं
```bash
streamlit run app.py
```

## 🌿 आयुर्वेदिक आधार (Ayurvedic Foundation)

इस ऐप का डेटा निम्न ग्रंथों पर आधारित है:

- **चरक संहिता** (Charaka Samhita) — शारीरस्थान
- **सुश्रुत संहिता** (Sushruta Samhita) — शारीरस्थान  
- **अष्टांग हृदयम्** (Ashtanga Hridayam) — शारीरस्थान
- **काश्यप संहिता** (Kashyapa Samhita) — बाल रोग
